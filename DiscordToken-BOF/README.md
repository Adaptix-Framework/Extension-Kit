# DiscordToken-BOF

Discord token finder as a Beacon Object File (BOF) for the [Adaptix](https://github.com/adaptix-framework/AdaptixC2) framework.

Scans LevelDB storage for Discord, Discord Canary, Discord PTB, and Lightcord. Decrypts tokens using DPAPI + AES-256-GCM.
